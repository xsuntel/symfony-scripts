# Documentation Rules

## Documentation

- **README.md**: Keep the main README updated with project structure and setup instructions.
- **Scripts**: Ensure all scripts have accompanying markdown documentation if complex.

## Diagrams

- **Location**: Store diagrams in `diagram/`.
- **Format**: Use standard formats (e.g., .drawio, .png, .svg).

## Project

### Cache - Redis

* Connect

```bash
redis-cli -h 127.0.0.1 -p 6379
```

* Information

```bash
redis-cli info
```

* help

```bash
redis-cli help
```

* Monitoring

```bash
redis-cli monitor
```

#### CRUD

* Keys

```bash
keys *{keyword}*
```

* Value


```bash

```

* Delete
